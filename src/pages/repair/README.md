# Repair System Pages

All Repair Management System pages are located here.

## Pages

- `RepairAll.tsx` - All repair tasks list
- `RepairCreate.tsx` - Create new repair task
- `RepairCheckAll.tsx` - All repair check tasks
- `RepairCheckPending.tsx` - Pending repair checks
- `RepairCheckHistory.tsx` - Repair check history
- `SentToVendorAll.tsx` - All sent to vendor tasks
- `SentToVendorPending.tsx` - Pending sent to vendor
- `SentToVendorHistory.tsx` - Sent to vendor history
- `StoreInAll.tsx` - All store in tasks
- `PaymentPending.tsx` - Pending payments
- `PaymentHistory.tsx` - Payment history

## API Usage

All pages use `repairApi` from `../../services/repairApi`
