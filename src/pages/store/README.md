# Store System Pages

All Store Management System pages are located here.

## Pages

- `StoreIndentCreate.tsx` - Create store indent
- `StoreIndentPending.tsx` - Pending store indents
- `StoreIndentHistory.tsx` - Store indent history
- `IndentAll.tsx` - All indents
- `IndentSubmit.tsx` - Submit indent
- `PoPending.tsx` - Pending purchase orders
- `PoHistory.tsx` - PO history
- `Items.tsx` - Items list
- `Stock.tsx` - Stock information
- `Uom.tsx` - Unit of measure
- `CostLocation.tsx` - Cost locations
- `VendorRatePending.tsx` - Pending vendor rate updates
- `VendorRateHistory.tsx` - Vendor rate history
- `ThreePartyPending.tsx` - Pending three party approvals
- `ThreePartyHistory.tsx` - Three party approval history

## API Usage

All pages use `storeApi` from `../../services/storeApi`
